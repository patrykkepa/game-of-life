## Usage & development setup
> Prerequisites: get yourself a C++ compiler e.g. G++.
1. Clone/pull repository.
2. In source folder run compiler
```sh
$ g++ main.cpp Engine.cpp Matrix.cpp PrinterTXT.cpp -o main.exe
```

3. Run the program

```sh
$ main.exe
```
